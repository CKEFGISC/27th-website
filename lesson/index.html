建電大社課
想要學習程式卻遲遲無法下手嗎？想要趕快學會程式卻覺得老師進度太慢嗎？
建電大社課傳統上會帶著大家一步一步從零開始一起學習C++這一門語言。相較於小社課，大社會以較慢速度更仔細的走過整個基礎的C++語法，不用害怕會跟不上腳步。
另外，今年我們也在靠近期末時加入了資料結構手刻的課程，給予大家練習語法的機會。上完這些內容，保證你資訊課時完全可以躺平的啦～
北資大社課
北一資研的大社課會由北一女中資訊老師何雪溱老師授課，上課內容會從C++語法到演算法，每個單元都會提供題單自由練習，上課時也會提供競賽資訊。在大社課中學習到的較像是平常資訊課的內容，上完我們的大社課後，資訊就可以躺一年啦～
建北電資聯合放學小社課
演算法
演算法小社著重在競賽中會使用到的各種語法、資料結構與演算法。透過我們的課程，讓學弟妹對資訊感到有興趣，因而踏入競賽程式的領域。當你學完整學期社課，要進入校隊、取得APCS好成績，絕對不會有問題！
python
身為高一新生的你，是不是對於程式設計感到特別有興趣，卻又不知從何開始學習呢？加入我們建北電資大家庭，就可以學到python這門對新手極為友善的程式語言囉！python的程式極容易理解，編寫程式的過程也特別容易，就算是完全沒學過程式語言的人也能很快就學會！
網頁
每天在打開瀏覽器在網頁中穿梭的你，想不想要設計自己的網頁呢？總是套模板了無新意，快來網頁聯課學習如何從0開始製作網頁吧！在這堂課中可以學到基礎的網頁前端製作，排版、動畫一次到手，不用複雜的邏輯推演，只要動手嘗試，就可以有豐富的作品產生，還可以分享給朋友！
專案建置
想不想要練習寫一個屬於自己的專案呢？想不想要學習一些在大型程式中常常需要的技巧與語法呢？想不想要學習Linux終端指令呢？在專案小社裡面你通通可以學到！我們會從環境調整的教學一步步帶領到物件導向程式設計，讓你可以自己寫出自己的專案！快來參加週三的專案建置小社喔！

北資中午小社課
python
想要入門程式語言嗎？來學現今2022最熱門的程式語言-python準沒錯！
Python的程式碼淺顯易懂，保證沒經驗的新手也能立即學會；而且功能也很強大，函式庫也很多，許多東西不用自己手刻~
在課堂中，我們會從最基礎的語法教起，從輸入輸出、變數到list都會教到，也會有大量題目可以練習熟悉語法！
心動不如行動，學妹來上python小社啦
C++解題
想要找一個專門練習C++的時間嗎？每次解題debug都找不到方向但沒有人可以問嗎？那就來星期二的小社課和我們一起解題吧！ 因為C++超快的運行速度，是如今程式競賽中最常被運用的語言，在解題過程中遇上任何問題，學姊都會盡全力幫忙喲！✧◝(⁰▿⁰)◜✧
網頁
有沒有學妹想要擁有自己的部落格？或是設計屬於自己的網頁？不同於競賽程式，網頁不僅用途廣，也可以很快的產出作品與成就感，更可以展現自己的美感與創意、與他人分享！
unity
你喜歡玩遊戲嗎?想知道熱門遊戲是如何製作而成的嗎?或是，想不想自己做一款遊戲? Unity小社課是以撰寫遊戲程式碼與學習軟體操作，製作出遊戲為目標的課程，學習與C++類似的C#語法，並在unity內執行程式。從最基礎的角色行走，到架設整個遊戲的循環都要自己動手寫程式碼來控制，課程偏向基礎與多元化。
